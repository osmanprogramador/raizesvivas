# 🌱 Raízes Vivas

**Preservação de Memória, Transparência Agroecológica e Inclusão Digital Rural.**

O projeto **Raízes Vivas** é um ecossistema digital desenvolvido em Flutter para fortalecer comunidades rurais e assentamentos sustentáveis. Através de tecnologia de ponta, buscamos registrar a história local, valorizar a produção agroecológica e promover o turismo pedagógico.

---

## 🚀 O Problema e a Solução

A maioria das comunidades rurais sustentáveis possui uma riqueza imensa de conhecimento (história, técnicas de plantio, biodiversidade) que se perde por falta de registro.

**O Raízes Vivas resolve isso através de:**
1.  **Inclusão Digital:** Traz tecnologia para o campo, capacitando jovens moradores como gestores de tecnologia.
2.  **Transparência Agroecológica:** Permite que o consumidor final veja a origem ética de seu alimento via georreferenciamento e fotos.
3.  **Memória e Patrimônio:** Registra permanentemente a história da comunidade, transformando o assentamento em um polo de turismo consciente.

## 🛠️ Tecnologias Principais

*   **Mobile & Web:** [Flutter](https://flutter.dev) (Android, iOS e Web)
*   **Backend:** Firebase (Firestore, Cloud Storage, Auth)
*   **Infraestrutura:** Conectividade via Starlink
*   **Interação:** QR Codes físicos para acesso direto a conteúdos geolocalizados.

## 📱 Funcionalidades

- **Exploração por QR Code:** Aponte a câmera para as placas físicas no assentamento e descubra histórias, fotos e vídeos sobre o local.
- **Mapeamento de Pontos de Interesse:** Navegação por GPS para trilhas, hortas e centros comunitários.
- **Gestão de Conteúdo:** Painel administrativo para curadoria e publicação de memórias e dados agroecológicos.
- **Funcionamento Offline:** Sincronização inteligente para áreas com conexão limitada.

## 🏗️ Como Executar

1.  **Pré-requisitos:**
    - Flutter SDK (versão >= 3.0.0)
    - Conta no Firebase

2.  **Configuração:**
    ```bash
    # Clone o repositório
    git clone https://github.com/osmanprogramador/raizesvivas.git

    # Entre no diretório
    cd raizesvivas

    # Instale as dependências
    flutter pub get
    ```

3.  **Rodar o App:**
    ```bash
    flutter run
    ```
